# Totoro-paradise 改进版

> Fuck `totoro school`, without MITM.

The name is netaed from `nekopara`.

---

## ✨ 改进功能

相较于原版的主要改进:

- ✨ **新增自定义跑步时长功能** - 可精确到秒
- ✨ **支持分钟和秒数分别输入** - 更灵活的时间控制
- 🐛 **修复自定义时间导致的卡死问题**
- 📚 **添加启动脚本** - `start.sh` 和 `launch-app.sh`
- 📚 **添加使用说明文档** - 中文使用指南

## 🏗️ How to build

```bash
pnpm i
pnpm build
```

## 🚀 How to run

### 方式一: 使用启动脚本 (推荐)

```bash
bash start.sh
```

### 方式二: 直接运行

```bash
pnpm dev      # 开发模式
pnpm start    # 生产模式
```

## ⚛️ How to develop

```bash
pnpm dev
```

访问地址: http://localhost:3000
