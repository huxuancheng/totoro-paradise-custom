# Totoro-paradise 使用说明

## ✅ 项目已克隆并安装依赖

**项目位置**: `~/Desktop/totoro-paradise`

## 🚀 快速开始

### 1. 进入项目目录
```bash
cd ~/Desktop/totoro-paradise
```

### 2. 启动开发模式
```bash
npx pnpm dev
```

项目将在 `http://localhost:3000` 启动

### 3. 构建项目
```bash
npx pnpm build
```

### 4. 运行构建后的项目
```bash
npx pnpm start
```

## 📝 可用命令

| 命令 | 说明 |
|------|------|
| `npx pnpm dev` | 开发模式（热重载） |
| `npx pnpm build` | 构建生产版本 |
| `npx pnpm start` | 运行构建后的项目 |
| `npx pnpm generate` | 生成静态站点 |
| `npx pnpm preview` | 预览构建结果 |

## 📁 项目结构

```
totoro-paradise/
├── components/       # Vue 组件
├── composables/      # Vue 组合式函数
├── pages/           # Nuxt 页面
├── plugins/         # Nuxt 插件
├── server/          # 服务端代码
├── src/             # 源代码
├── utils/           # 工具函数
├── app.config.ts    # 应用配置
├── nuxt.config.ts   # Nuxt 配置
├── Dockerfile       # Docker 镜像
└── docker-compose.yml  # Docker Compose
```

## 🛠️ 技术栈

- **Vue 3** - 前端框架
- **Nuxt 3** - 全栈框架
- **Vuetify** - UI 组件库
- **UnoCSS** - 原子化 CSS
- **TypeScript** - 类型安全
- **高德地图 API** - 地图服务

## 🐳 Docker 部署

```bash
cd ~/Desktop/totoro-paradise
docker-compose up -d
```
