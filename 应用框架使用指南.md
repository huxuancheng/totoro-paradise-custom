## 📋 应用框架菜单

### 1. 开发模式
启动热重载开发服务器
- 访问: http://localhost:3000
- 功能: 代码修改自动重载
- 停止: Ctrl+C

### 2. 构建项目
构建生产版本
- 生成: `.output/` 目录
- 输出: 优化后的静态文件

### 3. 运行生产版本
运行构建后的项目
- 访问: http://localhost:3000
- 特点: 性能优化版本

### 4. 生成静态站点
导出静态 HTML 文件
- 输出: `.output/public/`
- 用途: 静态网站部署

### 5. 预览构建
预览构建结果
- 用于测试生产构建
- 快速验证发布版本

### 6. 清理缓存
清理项目缓存文件
- 删除: `.nuxt/`
- 删除: `.output/`
- 删除: `node_modules/.cache/`

### 7. 检查状态
查看项目详细信息
- Node.js 版本
- npm/pnpm 版本
- 已安装包数
- 构建状态
- Git 信息

### 8. 查看日志
查看最近的构建日志
- npm 构建日志
- 错误信息

## 🎯 快速开始

### 首次使用

```bash
# 1. 进入项目目录
cd ~/Desktop/totoro-paradise

# 2. 启动开发模式
~/Desktop/totoro-paradise/start.sh

# 3. 选择 1（开发模式）
```

### 日常开发

1. 双击桌面图标打开应用框架
2. 选择"开发模式"
3. 在浏览器访问 http://localhost:3000
4. 开始开发

### 发布项目

1. 在应用框架选择"构建项目"
2. 构建完成后选择"运行生产版本"
3. 测试确认无误
4. 部署到服务器
